## About The Project

A fun twist on the classic game. Given a tagline, can you guess the movie?

This project was built with Python and its Pandas library, and The Movie Data Base data.


<!-- GETTING STARTED -->
## Getting Started

To enjoy this game, please clone this repository to your local machine and run in your terminal.

Use command:

  ```python3 tagline.py```


### Prerequisites

You will need python 3 installed to run the game. This game has not been tested on Python 2

### Installation

To install game clone repository to your local machine

To install python go to: https://www.python.org/downloads/ and follow the instructions for your specific operating system
